# Next.js 15 Product App

## Features
- Landing page with navbar, hero, highlights, footer
- Google login with NextAuth.js
- Public product list & details
- Protected "Add Product" page (only logged-in users)
- Products stored in memory / JSON

## Setup
```bash
git clone <repo-url>
cd my-app
npm install
cp .env.local.example .env.local
npm run dev
```

Open: [http://localhost:3000](http://localhost:3000)
