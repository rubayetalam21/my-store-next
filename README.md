# My Store - Next.js E-commerce Project

## Project Description
My Store is a modern e-commerce web application built with **Next.js**, **Tailwind CSS**, **MongoDB**, and **NextAuth.js** for authentication. Users can browse products, view product details, and log in using **Google OAuth**. Admin users can add new products via a protected interface. The project is deployed on **Vercel**.

---

## Features
- Browse and view products dynamically from MongoDB.
- Detailed product pages with image, description, and price.
- Admin dashboard to add new products.
- Google OAuth login via NextAuth.js.
- Fully responsive and elegant design using Tailwind CSS.

---
